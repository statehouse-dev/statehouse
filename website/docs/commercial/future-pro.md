# Commercial Editions (Draft)

:::warning Draft / Subject to Change

This document describes potential future commercial editions of Statehouse. These are conceptual plans and not commitments. Features, pricing, and availability are subject to change.

:::

Statehouse is free to use in production.

## Free Edition

The current Statehouse release is free to use, including:

- Full source code access
- Production use without restrictions
- All core features:
  - Strongly consistent state storage
  - Transaction support
  - Event replay
  - Crash recovery
  - Python SDK
  - CLI tooling
- Community support via GitHub issues

## Statehouse Pro (Conceptual)

A potential future commercial edition may include:

- Priority support with defined response times
- Additional operational features:
  - Advanced monitoring and alerting
  - Performance tuning guidance
  - Backup and restore tooling
- Extended documentation and best practices
- Direct access to the development team

## Statehouse Enterprise (Conceptual)

A potential future commercial edition may include:

- All Pro features
- Additional enterprise features:
  - Authentication and authorization
  - Multi-tenant isolation
  - Advanced audit logging
  - Compliance reporting
- Dedicated support channel
- Custom integration assistance
- On-site deployment support

## Notes

These editions are conceptual and not yet available. No pricing has been defined. The existence of future commercial editions does not affect the free edition's availability or features.

If you have questions about commercial licensing or support, contact licensing@statehouse.dev.
